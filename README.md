## Usage:
`python3 compareTickers.py NUMDAYS_BEFORE_START NUMDAYS_BEFORE_END #TICKER_1 #TICKER_2 ... #TICKER_N`

## Example
This compares the last 30 days for AAPL, MSFT, and SPY

`python3 compareTickers.py 30 0 AAPL MSFT SPY`
